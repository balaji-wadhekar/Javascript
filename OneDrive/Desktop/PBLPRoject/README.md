# 🎓 College Grievance Redressal Portal

A modern, responsive web application for managing student complaints and grievances in educational institutions.

## ✨ Features

### For Students:
- 📝 File complaints with category selection
- 📊 View personal complaint statistics (Total, Pending, Resolved)
- 🔍 Search and filter complaints by status and category
- 🗑️ Delete pending complaints
- 📱 Responsive design for mobile and desktop

### For Administrators:
- 👀 View all complaints from all students
- 📈 Comprehensive statistics dashboard
- 📂 Category-wise breakdown of complaints
- ⚡ Update complaint status (Pending → In Progress → Resolved)
- 🔍 Advanced search and filtering options
- 📊 Real-time analytics

## 🚀 How to Run

1. **Download/Clone** this project to your computer
2. **Open** `index.html` in any modern web browser (Chrome, Firefox, Edge, Safari)
3. **Login** using demo credentials:
   - **Student**: `student@gmail.com` / Password: `123`
   - **Admin**: `admin@gmail.com` / Password: `admin`
